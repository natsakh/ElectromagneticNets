# Initial-Boundary-Value Problems

This folder contains analytical solutions for one-dimensional electromagnetic initial-boundary-value problems in time-varying media.

## Contents

- Time-switched half-space
- Time-switched dielectric slab

Each notebook contains:

- mathematical formulation of the problem;
- analytical derivation of the solution;
- Laplace-domain solution;
- inverse Laplace transform;
- Python implementation of the final equations;
- example demonstrating the analytical solution.
