# Initial-Value Problems

This folder contains analytical solutions for one-dimensional electromagnetic initial-value problems in time-varying media.

## Contents

- Abrupt temporal transition
- Temporal slab
- Continuous temporal transition (ramp)
- Sequential temporal transitions

Each notebook contains:

- mathematical formulation of the problem;
- analytical derivation of the solution;
- Python implementation of the final equations;
- example demonstrating the analytical solution.
